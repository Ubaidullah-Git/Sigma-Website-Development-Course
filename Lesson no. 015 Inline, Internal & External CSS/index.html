Inline CSS, while not ideal for all situations, is primarily used to apply a unique style to a single HTML element, providing specific control and the highest priority over other CSS types.

Internal CSS is mainly used when a single HTML page needs a unique style that shouldn't be applied to other pages on the website.

External CSS is primarily used for maintainability, consistency, and performance, especially on multi-page websites. Storing all styles in a single file allows efficient site design management.
